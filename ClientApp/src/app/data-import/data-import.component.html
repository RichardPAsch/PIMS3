- Data Import for portfolio -
  • Allows for populating the system with Revenue and/or Asset (Position) information.
  • Both Revenue & Asset data imports must follow required XLSX spreadsheet templated formats.
  • Revenue imports require the prior existence of a Asset/Position for each income record being processed.
  • Asset/Position imports via 'Asset [new]', require the existence of a Profile for each Asset/Position added.
  • Account types currently supported for data importing include: CMA, Roth-IRA, IRA, 401(k), & Keogh.
  • Asset/Position importing via 'Asset [new]' will automatically create any necessary Profile(s) as needed.
  • Any ticker to be added, may first be tested for automatic Profile creation via 'Asset Profile' lookup.
  • Any ticker to be added, NOT found to have a Profile, may be manually added via 'Asset Profile'.
